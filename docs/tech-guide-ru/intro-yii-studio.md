Что такое Yii Studio?
==============


![Yii Studio](images/tech-logo-yii-studio.png)

Yii Studio – это платформа для разработки php-приложений построенная с использованием высокопроизводительного фрейворка Yii 2!
Готовая быстрая и удобная система управления контентом (CMS). 
Честные ООП, MVC, модульная структура.
Верстка системы на CSS фреймворке Bootstrap, иконки Font Awesome, интерфейс панели управления на основе AdminLTE.
          
            
* Управление модулями (добавление, отключение, настройки модулей)
* Управление пользователями, доступ на основе ролей и разрешений (RBAC)
* Управление глобальными настройками системы
* Очистка кеша, пересоздание файлов ресурсов из панели управления
* Управление миграциями БД
* Выполнение консольных команд
* Поддержка режима "live-edit" - при включении режима большинство элементов публичной части, становятся редактируемыми
* Локализация
* Журнал событий
            

Для каких задач можно использовать Yii Studio?
------------------------------------------

Yii Studio отлично подойдет для интернет-магазина, сайта-визитки, может быть легко изменена под выполнение различных задач.


Требования к ПО и знаниям
-------------------------

Yii Studio требует PHP 5.4.0 и выше. Наилучшим образом Yii Studio работает на последней версии PHP 7. 
Для дальнейшей разработки с использованием Yii Studio потребуются знания php-фреймворка Yii2.
